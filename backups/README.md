# نسخ احتياطية مهمة

هذا المجلد يحتوي على نسخ احتياطية من قاعدة البيانات والبيانات المهمة.

## الاستخدام

قبل أي تعديل كبير، انسخ قاعدة البيانات هنا:

```bash
Copy-Item ../Joja.Api/joja.db "./backup-$(Get-Date -Format 'yyyy-MM-dd-HHmm').db"
```

## ملاحظات

- احتفظ بنسخة احتياطية قبل أي نشر
- احتفظ بنسخة احتياطية قبل تعديل البيانات
- يمكنك استرجاع أي نسخة بنسخها إلى `Joja.Api/joja.db`
